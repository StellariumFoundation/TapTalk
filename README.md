**TapTalk** is a modern, accessible Text-to-Speech (TTS) soundboard application designed to give a voice to those who need it. 

Built with a stunning "Cyberpunk" dark-mode aesthetic, it allows users to save custom phrases and speak them aloud with a single tap. It is optimized for speed, clarity, and ease of use.
